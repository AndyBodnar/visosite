# Viscosity Gaming Studio: Coming Soon

Coming-soon landing page for **viscositynetwork.com**.

Static site (single `index.html` + `assets/`), no build step. Deployed on Vercel from this repo and live at https://viscositynetwork.com.

## Edit points
- **Launch date:** the countdown target is the `LAUNCH = new Date("...")` line near the bottom of `index.html`.
- **Social links:** the Discord / X / YouTube anchors in `index.html` currently point to `#`.
- **Loading labels:** the `STEPS` array in `index.html` drives the cycling boot-loader text.
